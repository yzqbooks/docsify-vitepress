兼容 Vue #

你可以直接在 Markdown 文件里写 Vue 代码，它将被执行。我们可以用它写一些 Vue 的 Demo 或者示例代码。

首先，将 Vue2.x 或 3.x 添加到你的index.html文件中。

为你的站点选择合适的生产版本或开发版本，以获得有用的控制台警告和 Vue.js devtools 支持。
